🧠 Intelli-Credit
AI Powered Credit Decisioning Engine

Intelli-Credit is an AI-based credit evaluation platform that helps financial institutions analyze company financial data and make smarter loan decisions.

The system analyzes financial indicators, generates visual insights, and provides loan approval or rejection recommendations.

🎯 Features

AI-Based Risk Analysis for corporate loan applications

Financial Data Processing for company financial indicators

Interactive Dashboard with graphs and visual insights

Loan Recommendation Engine (Approve / Reject)

Credit Risk Score Generation

Secure Login System
